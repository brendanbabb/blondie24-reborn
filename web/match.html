What am I looking at?

Both Paper-strict and Enhanced Anaconda networks are loaded and play each other at the chosen depth. Each side's evaluation of the current position is shown so you can see when they disagree about who's ahead — that's where the strength gap lives.

Default depth is 4 to keep the per-move search snappy. Crank it up to 6 for closer-to-real play (each move will pause longer while the AI thinks).

Random opening plies: minimax is deterministic — same position, same network, always picks the same move. Without randomness, every game from the standard starting position would be the exact same game replayed over and over. The "Random opening" setting plays N random legal moves before handing off to the AIs, so each game starts from a different position. Set it to 0 if you want the deterministic same-game-every-time behavior; otherwise 6–8 plies gives reasonable variety without skewing the outcome.
